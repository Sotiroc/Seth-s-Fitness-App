# Brand

The app should feel like a clean, practical workout tool.

## Current product context

The immediate target is a **web-first PWA** used on an iPhone home screen.

That means the brand should work well in:
- mobile Safari
- narrow phone widths
- standalone PWA mode

## Visual direction

- Material 3
- light mode default
- restrained use of color
- strong readability
- no noisy decoration

## Web-first UI note

For the current delivery:
- use letter avatars for exercises
- do not rely on uploaded thumbnails
- optimize for fast mobile use, not feature richness
