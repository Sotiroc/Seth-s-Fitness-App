# Fitness App

A personal workout tracker built in Flutter.

## Current goal

This project is currently focused on shipping a **web-first PWA** that can be used on an iPhone via Safari "Add to Home Screen".

## Current priorities

- working browser runtime
- local browser persistence
- simple one-user workflow
- same-day usable delivery

## Deferred for later

- native iOS/Android release
- store submission
- Supabase sync
- authentication
- exercise thumbnail uploads
